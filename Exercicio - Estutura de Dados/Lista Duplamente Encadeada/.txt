                    Lista Duplamente Encadeada
                    
    • Estrutura que armazena um conjunto de elementos, em
uma determinada sequência.
    • Permite alocação dinâmica de memória.
    • É constituída por elementos que possuem uma estrutura
composta de valor e endereço do elemento anterior e
do próximo elemento.
    • Se estivermos no primeiro elemento da lista, o campo
para o endereço do elemento anterior terá como valor:
NULL
    • Se estivermos no último elemento da lista, o campo para
o endereço do próximo elemento terá como valor: NULL
